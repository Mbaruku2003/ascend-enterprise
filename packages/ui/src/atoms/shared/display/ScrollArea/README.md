# ScrollArea

A low-level scrolling primitive used throughout Ascend UI.

## Features

- Vertical scrolling
- Horizontal scrolling
- Both axes
- Native browser scrolling
- Theme-ready
- Built on Box

## Usage

```tsx
<ScrollArea>
  ...
</ScrollArea>
```

```tsx
<ScrollArea orientation="horizontal">
```

```tsx
<ScrollArea orientation="both">
```
