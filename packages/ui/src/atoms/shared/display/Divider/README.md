# Divider

A low-level visual separator for grouping related content.

## Features

- Horizontal and vertical orientations
- Solid, dashed, and dotted variants
- Decorative mode for purely visual separation
- Accessible by default (`role="separator"`)

## Examples

```tsx
<Divider />

<Divider orientation="vertical" />

<Divider variant="dashed" />

<Divider decorative />
```
